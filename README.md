# longvinter-windows-installer
PowerShell installer script for Longvinter Windows Server

### Requirements
1. Administrator privileges
2. Windows 10+
3. PowerShell 5+ (tested with 5.X and 7.2)
4. Steam
5. Git + git-lfs

## Installation
```powershell
.\Manager.ps1 getkey
.\Manager.ps1 update
.\Manager.ps1 backup
.\Manager.ps1 uninstall```
